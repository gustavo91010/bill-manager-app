## Introdução

Bem-vindo ao projeto **bill-manager-app**! Este repositório é dedicado ao desenvolvimento da interface do usuário para um aplicativo de gerenciamento de contas. 
Ele será meu ponto de partida para aprender e praticar minhas habilidades no Front.

## Objetivo

O principal objetivo deste projeto é me ajudar a aprender enquanto crio uma interface amigável e funcional para a aplicação de gerenciamento de contas. Além disso, este projeto me dara a oportunidade de:

- **Aprender HTML, CSS e JavaScript:** E as tecnologias básicas do desenvolvimento web.

- **Praticar boas práticas de design:** Terei a chance de aplicar princípios de design de interface de usuário (UI) e experiência do usuário (UX) para criar uma interface atraente e intuitiva.

- **Integrar com APIs:** Poderei me comunicar com API para buscar e exibir dados relevantes na interface do usuário.

- **Ganhar experiência com controle de versão:** Aprender a usar o Git para controlar as mudanças em meu código.
